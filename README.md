### 기능 설명

- 입금액을 입력하여 음료를 선택해 카트에 담을 수 있습니다.
- 획득 버튼을 통해 획득한 음료는 리스트에 나타나며 또한 총 금액도 표시가 됩니다.
- 음료는 최대 5개까지이며 음료가 다 소진될 경우 '품절' 라벨이 표시되어 해당 음료수는 뽑을 수 없습니다.
